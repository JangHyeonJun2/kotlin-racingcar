### 입력
- 자동차 이름
- [x] ,(comma)로 스플릿
- [x] 이름의 글자수는 1 이상
- 시도 횟수
- [x] 숫자로만 입력

### 출력
- 실행 결과
- [x] 모든 시도의 결과를 가공해 출력
  - [x] 한 시도에 대해 문자열 가공
- 최종 우승자
- [x] 우승자 리스트를 ,(comma)로 가공

### 도메인
- [x] 임의의 숫자 생성
  - [x] 0에서 9 사이로 생성
- 자동차 경주
  - [x] 모든 자동차에 대해 전진 수행
    - [x] 자동차 전진
      - [x] 전진 기록 저장
- [x] 자동차 거리 별로 정렬
- [x] 우승자 결정
  - [x] 가장 멀리 이동한 모든 자동차 판별
